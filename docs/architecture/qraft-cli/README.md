# QraftWorx CLI Architecture Documents — Moved

The architecture, design, and execution documents for the QraftWorx CLI project have been moved to the [qraftworx-cli repository](https://github.com/coetzeevs/qraftworx-cli).

## Where to find them

**Repository:** [github.com/coetzeevs/qraftworx-cli](https://github.com/coetzeevs/qraftworx-cli)
**Path:** `docs/architecture/`

## What was here

These documents were created during the architecture and design phase of QraftWorx CLI while it was being planned in the Cerebro repo. They were copied to the qraftworx-cli repository during Phase 0 (project bootstrap) and are now maintained there.

Documents included:
- `design/qraft-architecture.md` — primary architecture specification
- `design/qraft-cli-review.md` — architect + tech lead critique
- `design/qraft-cli-support.md` — original proposal (historical)
- `design/nvs-feature-spec.md` — Nexus Vision System feature spec
- `design/nvs/` — NVS detailed architecture and ADRs
- `execution/implementation-plan.md` — TDD phase plan
- `execution/engineering-standards.md` — CI, testing, tooling standards
- `execution/repo-bootstrap.md` — repository setup guide
